usage:
type from console as below;
python Sousouwa_fetch.py <your desired short-story's URL>

example:
python Sousouwa_fetch.py 'http://coolier.dip.jp/sosowa/ssw_l/1/1057410215'



usage of Sousouwa_fetch_list.txt:
After write URL of sousouwa on URL_list.txt, type from console as below;
python Sousouwa_fetch.py
